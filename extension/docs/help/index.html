CCW 扩展开发指南

了解 CCW 扩展注册方式与 Blocky 高度自定义化能力

📋 扩展注册方式

CCW 扩展通过唯一的扩展 ID 进行注册。每个扩展在加载时需要声明其 ID、名称、颜色方案以及积木定义。 CCW 支持标准的 Scratch 扩展注册格式，同时提供了增强的 API 用于与平台深度集成， 包括事件广播、变量监听、角色控制等高级功能。扩展可以通过 URL 加载或从扩展仓库直接引用。

🧩 Blocky 的高度自定义化

CCW 的 Blocky 系统提供了极高的积木自定义能力。你可以自由定义积木的形状、颜色、输入类型和参数结构。 支持报告值积木、命令积木、C 型分支积木等多种类型。通过 Blocky 编辑器， 你可以可视化地设计积木外观，定义输入参数的类型与默认值，甚至创建可扩展输入的动态积木， 让扩展的交互体验更加灵活和强大。
